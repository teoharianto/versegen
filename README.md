# Verses Generator

Web React dengan Tailwind CSS bertema futuristik. Fitur:

- Landing page fullscreen dengan tombol Enter dan animasi pintu
- Dashboard utama menampilkan ayat Alkitab acak
- Renungan harian dengan pilihan tanggal dan date picker
- Admin untuk input renungan berbasis Markdown

## Menjalankan proyek

1. Install dependencies:
   - Jalankan `npm install`
2. Jalankan dev server:
   - Jalankan `npm run dev`

## Catatan

Data renungan disimpan di localStorage browser. Hapus localStorage untuk reset data.
